## Requirements

The following is necessary to build this clock:

| amount | item |
|-:|:-|
| 1 | ESP8266 module |
| 25 | WS2812 RGB LED Strip |
| 1 | voltage regulator 3,3V |
| 1 | voltage regulator 5,0V |
| 1 | power supply |
| 1 | image frame 20x20cm (e.g. IKEA Ribba) |

## Libraries

not included:
